ODC EDITOR

ODC 编辑器

基于 GB/T 45312—2025 创建标准化 ODC 声明。所有数据存储在浏览器本地，可导出 JSON 后通过 PR 提交。

① 基础信息
② 编辑步骤
  1. 从下方层级树中点击任意元素，即可加入声明并设置「允许 / 不允许」
  2. "不允许"必须指定退出行为（抑制激活 / 触发退出 / 两者皆有）
  3. 「允许」时若有量化参数（速度、坡度、能见度等）请填入参数范围
  4. 右侧实时预览生成的 JSON。完成后下载 JSON / Markdown
  5. 把 JSON 文件放入 data/examples/ 提交 PR 到 GitHub 仓库
